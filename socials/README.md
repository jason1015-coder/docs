# Social cadence templates

Drafts for the social posts that go out at each pipeline event. One per event. Replace `{...}` placeholders before posting. Voice should match the rest of NC: operational, understated, honest. See the [Brand Guidelines](/collective/organisation/brand) for the full voice spec.

These are templates, not contracts. Adapt to the platform and to what actually shipped.

Pipeline events covered:

- `whitepaper-merged.md` — a new whitepaper has been published, public review window opens.
- `review-closing.md` — five days before a review window closes, the last call to comment.
- `build-approved.md` — Stage 3 yes, project moves to build.
- `v0.1-shipped.md` — project has hitv1.0.
- `declined-or-paused.md` — honest decline / pause posts. Optional but high-value.

When NC eventually has its own platform-specific templates (LinkedIn long form, X thread shape, Discord announcement format), those can live as siblings to these.
